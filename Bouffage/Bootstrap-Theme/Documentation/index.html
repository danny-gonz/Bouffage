SocialChef - Social Recipe HTML Template

SocialChef is a beautiful, clean and responsive template that comes with advanced search and front-end submissions as well as community features such as blogs, member profiles and activity feeds. Want to feature chefs and recipes? You got it! This template will definitely help you build a robust and lively food social community quickly.
But hey, you are not limited to recipes and food, you can use it for just about anything.

Installation

SocialChef is a HTML template so there's no real "installation" required, just unzip the main file you've downloaded from templateforest and after unziping you have your all template files ready to be used.


Folder structure

  • css folder: contains css stylesheet files. In these files it is defined how to display HTML elements.
  • images folder: contains all necessary images for the website.
  • js folder: contains javascript files. These files provide enhanced user experience and dynamic effects.

HTML structure

Head structure

The <head> element is a container for all the head elements. The head section contains elements that define the page description, keywords, and other SEO meta information. It also includes links to the stylesheets that define fonts, animations and styling of the page.

To edit the title, description or keywords of your page, open index.html or any other page for that matter and edit the text within the "content" element:

Css

There are two CSS files in this template. Animate.css is for animation purposes and has no other effects on the website. We will take a look at the style.css file here. The The first few lines are generic reset file. Many browser interpret the default behavior of html elements differently. By using a general reset CSS file, we can work round this. These lines also contains some general styling, such as anchor tag colors, font-sizes, etc. Keep in mind, that these values might be overridden somewhere else in the file.

Headings

HTML Headings are used to denote a title, emphasize a topic or give a summary to the main content. You can use headers in any context or area within the template. To create headings use the following html code:

<h1>This is a H1 Header</h1>

List Styles

To create a default unordered list with bullet, use the following Html code:

<ul>
<li>List item1</li>
<li>List item2</li>
<li>List item3</li>
</ul>

To create a default ordered list with numbers, use the following Html code:

<ol>

<li>List item2</li>

Scroll To Top

It is a button on the bottom of the page that scrolls the user back to top of the page. If you want to change the speed of scrolling simply change the number (in miliseconds).

				$(document).ready(function(){
					$('.scroll-to-top').click(function () {
						$('body,html').animate({
							scrollTop: 0
						}, 800);
						return false;
					});
				});
